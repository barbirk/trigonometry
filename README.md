# TrigPath (Académie Trigonométrique)

**TrigPath** is a self-paced, interactive Progressive Web Application (PWA) designed to teach trigonometry on triangles to secondary school students. Designed with a clean, high-contrast scientific journal aesthetic, it employs active learning and spaced repetition to build true mathematical intuition. 

Features English and French localization. Designed explicitly for tablets.

## Tech Stack
- Frontend: React 18, Vite
- Styling: Tailwind CSS V4, Custom CSS Variables
- Routing: React Router v6
- State Management: Zustand 
- PWA Support: `vite-plugin-pwa`
- Math Support: KaTeX 
- Internationalization: `react-i18next`

## Deploy 

This application is built as a static client-side PWA. Push to the `main` branch to trigger GitHub Actions which deploy the compiled application to GitHub Pages.

## Development

```bash
npm install
npm run dev
```
